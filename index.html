Chord Builder

Acessos: Repositório & Site

Esse pequeno projeto apresenta uma interface em que usuário pode escolher a posição das notas em uma cifra de violão e então encontrar o nome do acorde associado.

Foi um desafio muito interessante, pois a ideia era diferente do escopo de coisas que estou acostumado a implementar (CRUD, etc).
O projeto possui algumas limitações: a responsividade do site é limitada e o usuário só pode salvar seus acordes no local storage.

Tecnologias: Empacotador Web Parcel + Javascript + HTML/CSS + Biblioteca Tonal.js
